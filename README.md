Este repositório contém a política de privacidade oficial do **Caramelo's Bot**, um bot do Discord especializado em criar embeds personalizados.

## 🔍 O que é coletado?

O Caramelo's Bot coleta **apenas dados essenciais**:
- IDs de usuário do Discord
- IDs de servidores
- Comandos e interações com o bot

**Nós garantimos:**
- ✅ Nenhuma venda de dados
- ✅ Sem compartilhamento com terceiros
- ✅ Dados usados apenas para funcionamento
