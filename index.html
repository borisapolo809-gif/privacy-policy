📌Sobre o Caramelo's Bot

O Caramelo's Bot é um bot do Discord especializado em criar embeds de forma simples e intuitiva. Nosso objetivo é facilitar a criação de mensagens visuais e personalizadas para sua comunidade.

🎯 Função

Criar embeds personalizados

🌐 Plataforma

Discord

✅ Status

Ativo e em desenvolvimento

📊Quais Dados Coletamos?

O Caramelo's Bot coleta apenas as informações necessárias e essenciais para funcionar corretamente no Discord.

📝 Dados Coletados:

  • IDs de Usuário - Identificadores únicos do Discord para reconhecer quem usa o bot
  • IDs de Servidor - Identificadores dos servidores Discord onde o bot está instalado
  • Mensagens Relacionadas ao Bot - Comandos e interações diretas com o bot
❌ O que NÃO coletamos:
  • Senhas ou informações sensíveis
  • Dados bancários ou de pagamento
  • Histórico de mensagens gerais do Discord
  • Informações pessoais sem necessidade

💾Como Usamos os Dados?

Todos os dados coletados são utilizados exclusivamente para:

  1. Operar o bot - Permitir que o bot funcione corretamente
  2. Processar comandos - Executar os comandos que você solicita
  3. Criar embeds - Gerar as mensagens personalizadas que você deseja
  4. Melhorar o serviço - Entender como o bot é usado para melhorá-lo

🚫 O que fazemos com os dados:

✅ Nós GARANTIMOS que:
  • Nenhuma venda de dados - Seus dados nunca serão vendidos
  • Sem compartilhamento - Os dados não são compartilhados com terceiros
  • Sem publicidade direcionada - Não usamos seus dados para anúncios
  • Apenas necessário - Coletamos apenas o mínimo necessário

🔐Segurança dos Dados

A segurança das suas informações é nossa prioridade. Implementamos medidas de proteção para garantir que seus dados estejam seguros.

🛡️ Medidas de Segurança:

  • Dados criptografados em trânsito
  • Acesso restrito aos dados
  • Sem armazenamento desnecessário
  • Conformidade com regulamentações de privacidade

⚖️Seus Direitos

Você tem direitos sobre seus dados. Dependendo da sua localização, você pode ter direito a:

  • 📋 Acessar seus dados - Saber quais dados temos sobre você
  • 🗑️ Solicitar exclusão - Pediremos para deletar seus dados
  • ✏️ Corrigir informações - Corrigir dados incorretos
  • 📤 Exportar dados - Receber seus dados em formato portátil

Para exercer qualquer desses direitos, entre em contato conosco através do Discord.
